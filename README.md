# Web Desktop Environment (2026)

This is a front-end web development project that simulates a desktop operating system inside a browser window.

## Features
*   **OOBE (Out of Box Experience):** Simulates a first-time setup wizard with local storage persistence.
*   **Window Manager:** Custom JavaScript logic to handle dragging, minimizing, and closing application windows.
*   **CSS Animations:** Features a "living" aurora background using pure CSS keyframes.
*   **Terminal Emulator:** A simulated command-line interface that accepts basic commands.
*   **Responsive Design:** Uses Flexbox and modern CSS variables.

## How to Run
This project is hosted via GitHub Pages. No backend installation is required. Simply open `index.html` in any modern web browser.

## Technologies Used
*   HTML5
*   CSS3 (Glassmorphism UI)
*   JavaScript (ES6)
